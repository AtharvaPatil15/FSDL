## Profile Website

A single-page personal portfolio (Hero, About, Education, Experience, Skills, Projects,
Contact) with a sticky responsive navbar, smooth-scroll navigation, dark/light theme
toggle, and scroll-reveal animations. All editable content lives in
[`profile-website/src/data.js`](./profile-website/src/data.js).

### Tech stack

- **React 18** (functional components + hooks)
- **Vite 5** (build tooling / dev server)
- **Tailwind CSS 3** (styling, CSS-variable theming)
- **Vercel** (hosting / deployment)

### Run locally

```bash
cd profile-website
npm install
npm run dev      # start the dev server
npm run build    # production build → dist/
```
